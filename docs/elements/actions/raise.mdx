# Raise Element

The `<raise>` element raises an internal event:

<DynamicCodeBlock lang="jsx" code={`<raise event="success" />`} />

<TypeTable
  type={{
    id: {
      description: 'Unique identifier',
      type: 'string',
      optional: true
    },
    event: {
      description: 'Event name to raise',
      type: 'string',
      optional: true
    },
    eventexpr: {
      description: 'Expression evaluating to the event name',
      type: 'Expression',
      optional: true
    }
  }}
/>

## Examples

### Basic Event Raising

<DynamicCodeBlock lang="jsx" code={`<raise event="complete" />`} />

### Dynamic Event

<DynamicCodeBlock lang="jsx" code={`<raise 
  eventexpr={({state}) => state.nextEvent} 
/>`} />

### State Transition

<DynamicCodeBlock lang="jsx" code={`<state id="processing">
  <onentry>
    <script>
      processData();
    </script>
    <raise event="processingComplete" />
  </onentry>
  <transition event="processingComplete" target="done" />
</state>`} />

## Usage Notes

- Use for internal state machine communication
- Events are processed immediately
- Can trigger state transitions
- Useful for workflow control
