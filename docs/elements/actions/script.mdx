## Examples

### Inline Script

<DynamicCodeBlock lang="jsx" code={`<script>
  // Perform calculations
  const result = calculateTotal(items);
  console.log(\`Total: \${result}\`);
</script>`} />

### External Script

<DynamicCodeBlock lang="jsx" code={`<script src="/scripts/utils.js" />`} />

### State Manipulation

<DynamicCodeBlock lang="jsx" code={`<script>
  // Update state based on calculations

## Usage Notes

- Use scripts for complex logic that's difficult to express with other elements
- Keep scripts focused and modular
- Consider moving complex logic to external files
- Be cautious with global state modifications
- Use appropriate error handling
- Document script functionality clearly

## Best Practices

1. **Code Organization**
   - Keep scripts small and focused
   - Use meaningful variable names
   - Comment complex logic
   - Follow consistent coding style

2. **State Management**
   - Use _dataModel for state access
   - Validate state changes
   - Handle edge cases
   - Clean up after operations

3. **Error Handling**
   - Use try/catch blocks
   - Log errors appropriately
   - Provide fallback behavior
   - Handle async operations properly
