# Assign Element

<Banner className="bg-blue-500 text-white rounded-lg">Coming soon: This feature is not yet available. Please check back soon.</Banner>


The `<assign>` element modifies the data model:

<DynamicCodeBlock lang="jsx" code={`<assign 
  location="state.count"
  action="increment"
/>`} />

<TypeTable type={{
    location: {
      description: "The data model location to modify",
      type: "string",
      required: true
    },
    action: {
      description: "Type of assignment operation (\"assign\" (default), \"increment\", \"decrement\", \"append\")",
      type: "string",
      required: false
    },
    expr: {
      description: "Expression to evaluate for the assignment",
      type: "Expression",
      required: false
    }
  }} />

## Examples

### Basic Assignment

<DynamicCodeBlock lang="jsx" code={`<assign location="state.name" expr={({userInput}) => userInput.name} />`} />

### Increment Counter

<DynamicCodeBlock lang="jsx" code={`<assign location="state.count" action="increment" />`} />

### Append to Array

<DynamicCodeBlock lang="jsx" code={`<assign 
  location="state.items" 
  action="append" 
  expr={({userInput}) => userInput.newItem} 
/>`} />

## Usage Notes

- Use clear location paths that reflect the data structure
- Consider using expressions for dynamic values
- Choose appropriate action types for the operation
- Validate data before assignment when necessary
